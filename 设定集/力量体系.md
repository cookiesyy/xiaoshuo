# 力量体系

## 修炼境界

```
炼气境（1-9层）     → 引气入体，淬炼肉身
筑基境（初/中/后/圆满） → 凝聚根基，灵力液化
金丹境（初/中/后/圆满） → 凝聚金丹，寿元500载
元婴境（初/中/后/圆满） → 元婴出窍，寿元1000载
化神境（初/中/后/圆满） → 神识化形，寿元2000载
炼虚境（初/中/后/圆满） → 虚空凝练，寿元5000载
合体境（初/中/后/圆满） → 天人合一，寿元10000载
大乘境（初/中/后/圆满） → 功德圆满，寿元100000载
渡劫境（1-9重天）    → 承受天劫，飞升在即
真仙境（人/地/天/金） → 成就仙位，超脱生死
```

## 战力对比

| 境界 | 基础战力 | 越级难度 |
|------|---------|---------|
| 炼气境 | 10 | 简单 |
| 筑基境 | 100 | 较难 |
| 金丹境 | 1000 | 困难 |
| 元婴境 | 10000 | 极难 |
| 化神境 | 100000 | 不可能 |

**注**：主角凭借吞噬血脉，可越1-2个大境界战斗

## 吞噬血脉能力

### 基础能力

| 能力 | 描述 | 限制 |
|------|------|------|
| **吞噬** | 吞噬敌人/妖兽/天材地宝 | 每日3次 |
| **提取** | 从吞噬物中提取能力/记忆 | 需要炼化时间 |
| **进化** | 吞噬同类型目标强化自身 | 上限限制 |
| **血脉技** | 随境界解锁血脉技能 | 需血脉觉醒度 |

### 血脉觉醒

| 觉醒度 | 解锁能力 |
|--------|---------|
| 10% | 吞噬基础能力 |
| 30% | 血脉技「吞噬之爪」 |
| 50% | 血脉技「吞噬领域」 |
| 70% | 血脉技「吞噬万物」 |
| 100% | 终极能力「吞噬苍穹」 |

### 吞噬上限

| 境界 | 可吞噬上限 |
|------|-----------|
| 炼气境 | 同境界妖兽 |
| 筑基境 | 金丹境妖兽 |
| 金丹境 | 元婴境妖兽 |
| 元婴境 | 化神境妖兽 |
| 化神境 | 炼虚境妖兽 |
| ... | ... |
