# 我和房东有个约会 - 总纲

## 核心设定

| 项目 | 内容 |
|------|------|
| 标题 | 我和房东有个约会 |
| 题材 | 都市言情 |
| 主角 | 叶凡（落魄富二代） |
| 女主 | 顾晚晴（冷艳房东） |
| 核心卖点 | 冷艳房东×痞帅租客的先虐后甜感情线 |
| 目标字数 | 150万字 |

---

## 卷纲

### 第一卷：相遇（第1-150章）

**主题**：租房初见，互相看不顺眼

**核心事件**：
1. 第1-30章：租房风波
   - 叶凡找工作屡次碰壁
   - 找房子时遇到顾晚晴
   - 顾晚晴对叶凡的第一印象：吊儿郎当，不靠谱

2. 第31-80章：入住冲突
   - 两人生活习惯不同，矛盾不断
   - 叶凡半夜打游戏、顾晚晴嫌吵
   - 互相吐槽，但不得不共处

3. 第81-150章：互相了解
   - 顾晚晴发现叶凡不是真的懒
   - 叶凡发现顾晚晴只是外冷
   - 两人关系从敌对变为微妙

**第一卷看点**：
- 欢喜冤家的日常互动
- 冷热性格的碰撞
- 埋下心动的种子

---

### 第二卷：磨合（第151-300章）

**主题**：日常相处，发现对方优点

**核心事件**：
1. 第151-200章：生活细节
   - 叶凡主动帮忙修东西
   - 顾晚晴生病，叶凡照顾
   - 两人关系缓和

2. 第201-250章：心动的瞬间
   - 某个小细节打动对方
   - 顾晚晴开始期待见到叶凡
   - 叶凡发现自己在意顾晚晴

3. 第251-300章：内心挣扎
   - 两人都意识到感情变化
   - 顾晚晴因为身份差距犹豫
   - 叶凡因为自尊心不敢表达

**第二卷看点**：
- 暧昧升温
- 内心戏丰富
- 欲言又止的互动

---

### 第三卷：心动（第301-450章）

**主题**：明确心意，但面临阻碍

   - 顾晚晴也有所察觉

2. 第351-400章：身份差距

   - 顾晚晴开始回避
   - 叶凡感到受伤

3. 第401-450章：短暂分离
   - 顾晚晴提出让叶凡搬走
   - 叶凡赌气同意
   - 两人分开后都开始不适应

**第三卷看点**：
- 告白被打断
- 身份差距的心理博弈
- 虐点开始出现

---

### 第四卷：虐心（第451-600章）

## 章节结构

每章2000-3000字，保持以下节奏：

```
单章结构：
- 开篇引入（300字）
- 中段发展（1500-2000字）
- 结尾互动（300字）

节奏控制：
- 每3章一个小互动
- 每10章一个重要情节
- 每30章一个剧情转折
```

---

## 核心法则

> **大纲即法律** - 遵循大纲，不擅自发挥
> **设定即物理** - 遵守设定，不自相矛盾
> **发明需识别** - 新实体由 AI 自动提取

---

## 写作风格

| 项目 | 要求 |
|------|------|
| 视角 | 第三人称限知视角，双线切换（叶凡+顾晚晴） |
| 节奏 | 先虐后甜，前半段以矛盾和心动为主，后半段甜蜜温馨 |
| 甜点 | 每3000字一个小互动，每10000字一个重要情节 |
| 虐点 | 身份差距、误会分离、家庭反对 |
